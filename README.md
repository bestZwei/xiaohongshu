# AI 聊天助手

一个简单的 AI 聊天界面，可以部署在 Cloudflare Pages 上。

## 功能特点

- 支持 Markdown 渲染
- 代码高亮显示
- 可自定义 API 设置
- 响应式设计
- 设置本地保存

## 部署说明

1. Fork 本仓库
2. 在 Cloudflare Pages 中连接仓库
3. 部署设置：
   - 构建命令：不需要
   - 输出目录：src
4. 完成部署

## 本地开发

直接打开 src/index.html 即可预览

## 注意事项

- 需要有效的 OpenAI API Key
- 建议使用自己的代理服务来中转 API 请求
